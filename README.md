A simple Prisma example using Apollo server.

## Requirements

- Node / NPM

## Usage

### Prepare

```
npm install
npm run migrate

cd web
npm install
```

### Run server

```
# In root directory
npm run dev
```

Access Playground at http://localhost:4000/

### Run client

```
# Inside /web directory
npm run serve
```

Access web client at http://localhost:8080/

### Build

```
npm run build
```
